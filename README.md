Features are
* uses iptables-persistent
* provides ipv4 & ipv6 support
* ossec hids
* ubuntu unattended upgrades
* sshd hardening

## Compatability

dda-pallet is compatible to the following versions
* pallet 0.8
* clojure 1.7
* (x)ubunutu14.04 / 16.04
